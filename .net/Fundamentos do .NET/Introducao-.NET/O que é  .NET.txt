.NET:
O .NET é uma plataforma de desenvolvimento unificado que permite a construção de sistemas e aplicações.
Uma plataforma de desenvolvimento unificado é um ambiente que reúne ferramentas, recursos e serviços para facilitar o desenvolvimento de software de forma integrada.

Diferença entre .NET Framework (Legado) e .NET:

Framework (.NET original) só funciona em windows;
.NET (ou .NET Core) é multiplataforma, a versão atual.

.NET Framework:
- Apenas Windows;
- WPF, Windows Forms e ASP.NET (4 e 5).

.NET:
- Windows, Linux e Mac - posso desenvolver em um SO e utilizar o código wm outro (que nem o JAVA);
- ASP.NET 5;
- Universal Windows APPs;
- .NET Native (vem da store, nativos do windows).

Ambos: 
- Runtime Components;
- Bibliotecas;
- Compiladores.
